# Odoo 17 ERP — Package L: Service Business

Advanced ERP package for service-oriented businesses, built on top of Package M (Retail/Distributor).

## Key Features

- **Accounting**: Auto GL, VAT reports (input/output), WHT (3%, 5%), trial balance, P&L, debit/credit notes, cash/bank documents, AR/AP ledger, payment approval, advance payments, outstanding debt tracking
- **Inventory Add-on**: Barcode / QR code stock management
- **Request for Quotation Add-on**: International purchasing with currency exchange
- **Dashboard & KPI**: Daily/monthly sales, cash flow monitoring, best-selling products, slow-moving stock alerts
- **Mobile App**: Owner dashboard, purchase approval, sales reports
- **Workflows**: Multi-level approval for quotations, purchase orders, PO, and expenses
- **POS**: Retail/restaurant ready, tablet support, real-time stock sync, instant tax invoices

## Includes all features from Package S & M:
- Sales Basic & Advance, Purchasing, RFQ, Inventory & Warehouse

## Tech Stack

- Python 3.10+
- Odoo 17 Community/Enterprise
- PostgreSQL
- XML (Views, Reports, Data)
